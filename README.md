## 📋 Общее описание

**AParts** — это современное веб-приложение для интернет-магазина автомобильных запчастей. Система работает полностью в браузере и не требует установки дополнительного программного обеспечения. Интерфейс построен на базе **Bootstrap 5**, что обеспечивает одинаково удобную работу как с персонального компьютера, так и с планшета или смартфона.

## ⚙️ Установка и запуск

### Требования
Перед началом убедитесь, что на вашем компьютере установлена **Java 17** или новее.

### Пошаговая инструкция

| Шаг | Действие | Описание |
|:---:|:---|:---|
| 1 | Скачайте файл `aparts.jar` | Из раздела Releases репозитория |
| 2 | Откройте терминал | В папке, куда скачан файл |
| 3 | Запустите приложение | Выполните команду: `java -jar aparts.jar` |
| 4 | Откройте браузер | Перейдите по адресу: `http://localhost:8080` |

## 🛍️ Работа с каталогом товаров

> Доступно для **всех пользователей**, включая гостей.

### Просмотр категорий
На главной странице представлена иерархическая структура категорий: Двигатель, Ходовая часть, Кузов, Трансмиссия, Электрика и другие.

### Поиск и фильтрация

| Элемент | Расположение | Как использовать |
|:---|:---|:---|
| Строка поиска | Верхняя часть экрана | Введите часть названия или артикул (например: `фильтр масл`) |
| Фильтр по бренду | Левая колонка | Выберите производителя из выпадающего списка |
| Фильтр по цене | Левая колонка | Укажите диапазон от и до |

### Пагинация
Внизу страницы каталога расположены кнопки перехода между страницами. На одной странице отображается до 20 товаров.

## 🔐 Регистрация и авторизация

### Регистрация нового пользователя
1. Нажмите кнопку **«Вход / Регистрация»** в правом верхнем углу.
2. Перейдите на вкладку регистрации.
3. Заполните поля: Имя, Email, Пароль.
4. Пароль должен содержать не менее 6 символов.
5. Нажмите **«Зарегистрироваться»**.

### Вход в систему
1. Нажмите **«Вход / Регистрация»**.
2. Введите Email и пароль.
3. Нажмите **«Войти»**.

### Восстановление пароля
1. Нажмите ссылку **«Забыли пароль?»** под формой входа.
2. Введите Email, указанный при регистрации.
3. Инструкция по сбросу придет на указанную почту.
4. Проверьте папку «Спам», если письмо не пришло.

## 👤 Личный кабинет клиента

После успешного входа в правом верхнем углу отобразится ваше имя.

### Профиль
Перейдите в раздел **«Мой профиль»** для редактирования:
- Имени и фамилии
- Контактного телефона
- Адреса доставки
- Пароля

### История заказов
В разделе **«Мои заказы»** отображается список всех оформленных вами заказов с указанием:
- Номера заказа
- Даты оформления
- Суммы
- Текущего статуса

## 🛒 Оформление заказа

### Пошаговая инструкция

| Шаг | Действие |
|:---:|:---|
| 1 | Добавьте необходимые товары в корзину |
| 2 | Перейдите в **Корзину** (иконка в правом верхнем углу) |
| 3 | Проверьте состав и количество товаров |
| 4 | Нажмите кнопку **«Оформить заказ»** |
| 5 | Укажите адрес доставки |
| 6 | Выберите способ оплаты |
| 7 | Подтвердите заказ |

После подтверждения заказу автоматически присваивается статус **«Новый»**.

---

## 📊 Отслеживание статуса заказа

В личном кабинете в разделе **«Мои заказы»** вы можете отслеживать текущий этап обработки:

| Статус | Значение |
|:---|:---|
| 🆕 **Новый** | Заказ поступил в систему, ожидает обработки менеджером |
| 🔄 **В обработке** | Сотрудник склада комплектует заказ |
| 🚚 **Передан в доставку** | Заказ передан курьеру или в транспортную компанию |
| ✅ **Доставлен** | Заказ получен покупателем |
| ❌ **Отменен** | Заказ отменен по инициативе клиента или из-за отсутствия товара |

## 🔧 Административная панель

> Доступно только для пользователей с ролью **ADMIN**

Администратор имеет доступ к скрытому разделу сайта `/admin`.

### Управление товарами
- Добавление новых позиций (название, цена, остаток на складе)
- Редактирование существующих карточек товаров
- Удаление товаров из каталога

### Управление персоналом
- Добавление учетных записей для работников склада
- Назначение роли **«Работник»**
- Деактивация учетных записей сотрудников

---

## 📦 Интерфейс работника склада

> Доступно только для пользователей с ролью **WORKER**

### Смена статусов заказа
Сотрудник склада видит список всех активных заказов и может изменять их статусы:

| Кнопка | Новый статус |
|:---|:---|
| **«Начать сборку»** | Новый → В обработке |
| **«Передать в доставку»** | В обработке → Передан в доставку |
| **«Подтвердить доставку»** | Передан в доставку → Доставлен |

## ❓ Часто задаваемые вопросы

<details>
<summary><b>Товар не добавляется в корзину</b></summary>
Проверьте сообщение об ошибке под карточкой товара. Вероятно, запрашиваемое количество превышает остаток на складе. Уменьшите количество и попробуйте снова.
</details>

<details>
<summary><b>Не приходит письмо для восстановления пароля</b></summary>
Проверьте папку «Спам» в вашей почте. Если письма нет и там, обратитесь к администратору системы.
</details>

<details>
<summary><b>Ошибка 404 — Страница не найдена</b></summary>
Вернитесь на главную страницу, нажав на логотип в левом верхнем углу.
</details>
